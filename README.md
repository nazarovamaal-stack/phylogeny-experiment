# Симуляционное исследование влияния положительного отбора на точность филогенетической реконструкции

## Описание

Данный проект реализует вычислительный пайплайн для изучения того, как положительный отбор на трансмиссивность вируса искажает топологию филогенетического дерева, построенного по нейтральным последовательностям. 

**Основные этапы:**
1. Генерация истинного дерева с отбором с помощью симулятора VGsim.
2. Добавление нейтральных мутаций на дереве с помощью AliSim.
3. Реконструкция дерева методом максимального правдоподобия.
4. Сравнение истинного и восстановленного деревьев с помощью двух метрик:
   - нормализованного расстояния Робинсона-Фолдса (RF) - чувствительно только к топологии,
   - взвешенного расстояния Робинсона-Фолдса (WRF) - учитывает также длины ветвей.
5. Статистическая обработка (U-тест Манна-Уитни) для оценки значимости различий между контрольной группой и группами с отбором.

**Варьируемые параметры:**
- Сила отбора `m` (0, 1, 5, 10)
- Число адаптивных сайтов `n_sites` (1, 2)
- Скорость нейтральных мутаций `µ` (0.0001, 0.0005, 0.001)

## Структура репозитория

- `config.py` - настройки эксперимента (параметры эпидемии, отбора, повторы и т.д.)
- `main.py` - основной скрипт, запускающий полный перебор комбинаций параметров.
- `simulation.py` - функции генерации истинного дерева и нейтральных последовательностей.
- `phylogeny.py` - запуск IQ‑TREE и расчёт метрик RF / WRF.
- `sequence.py` - вспомогательные функции для работы с FASTA.
- `system.py` - системные утилиты (выполнение команд, создание папок).
- `visualization.py` - сбор результатов, построение тепловых карт, boxplot, линейных графиков и выполнение теста Манна-Уитни.
- 

После запуска `main.py` в папках вида `exp_stable_<μ>/mX_selY_repZ` будут созданы файлы `result.txt` с вычисленными RF и WRF. Затем `visualization.py` объединяет их в единую таблицу `all_results.csv`, строит графики и сохраняет результаты теста в `mann_whitney_all.csv`.

## Требования

- Python 3.7 или выше
- VGsim (см. [репозиторий VGsim](https://github.com/Genomics-HSE/VGsim))
- IQ‑TREE (скачать с [официального сайта](http://www.iqtree.org))
- Установка Python‑библиотек (см. `requirements.txt`)
